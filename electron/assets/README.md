# Electron Assets

Place your application icons here:

- `icon.png` - PNG icon for development (256x256 or 512x512 recommended)
- `icon.ico` - Windows icon file for the .exe (use online converter or tools like IconConverter)

You can create icons using:
- Online tools: https://www.favicon-generator.org/
- GIMP/Photoshop
- Or use a placeholder icon for now

The icon is referenced in `electron/package.json` for the build process.
